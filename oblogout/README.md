# configuration for oblogout

## theme

### copy icons
``` shell
$ mkdir -p /usr/share/themes/ghreyt/oblogout
$ cp icons/* /usr/share/themes/ghreyt/oblogout/
```
### /etc/oblogout.conf

set theme in configuration

```
...
buttontheme = ghreyt
...
```

## set command

set proper command for each action like **oblogout.conf**
